Реализовать ping-pong c помощью библиотеки socket на Python. 
Клиент подключается к серверу и отправляет сообщение, сервер получает это 
сообщение и отправляет в ответ клиенту тоже самое сообщение. Реализация должна 
быть асинхронной, то есть несколько клиентов могут подключаться к серверу одновременно.
Рассмотреть реализацию с использованием генераторов в Python и библиотеки asyncio
